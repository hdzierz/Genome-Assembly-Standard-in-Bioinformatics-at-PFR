Naming Convention for Entities

Entities:
  • Contig (contig<zero padded integer>)
  • Scaffold (scaffold<zero padded integer>)
  • superscaffolds/chromosome (chr<integer>)
  • genes (<species abbreviation><zero padded interger>)

Naming Convention for File Names

  • FASTA: <species>[.cultivar].<assembly_build>.<sequence_type>.<id_type>.<id>.fa.gz
  • species: The systematic name of the species
  • assembly: The assembly build name
  • sequence_type: dna, dna_rm (masked), dna_sm (soft masked), rna
